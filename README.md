# nuxt-spanner-driver

## 実装予定のもの
- Insertができるように
- クエリが直接かける
  - 補完周りがしんどいのでGoogleのエディタでやってもらう方がいいかも
- 実行クエリのHistoryが見れる

## 準備
```bash
$ cp .env.example .env
# .envを編集したいspannerインスタンスの設定に変更
```
## 実行

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev
```

## ビルド
```bash
# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
